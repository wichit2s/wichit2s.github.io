Artificial Neuron
(Perceptron)

The Basic Building Block of Neural Networks

Structure

1
Inputs: features or data values (x₁, x₂, …)
2
Weights: importance assigned to each input (w₁, w₂, …)
3
Bias: adjusts the threshold for activation
4
Activation Function: introduces non-linearity (e.g. ReLU, Sigmoid)
5
Output: y = f( Σ (wᵢ·xᵢ) + b )

Perceptron
Visualization & Equations

Linear Combination + Bias → Activation

A perceptron computes a weighted sum of inputs plus a bias,
then applies an activation function:
y = f(Σ wᵢxᵢ + b).

Implementing a Perceptron in Python

Python + NumPy Implementation

import numpy as np

class Perceptron:
    def __init__(self, size, learning_rate=0.01):
        self.size = size
        self.learning_rate = learning_rate
        self.weights = np.random.randn(size)
        self.bias = 0.0

    def z(self, x):
        return np.dot(self.weights, x) + self.bias

    def f(self, x):
        return 1 if self.z(x) >= 0 else 0

    def fit(self, X, y, epochs=10):
        for _ in range(epochs):
            for xi, target in zip(X, y):
                pred = self.f(xi)
                error = target - pred
                self.weights += self.learning_rate * error * xi
                self.bias += self.learning_rate * error
Attributes: size, learning_rate
Function: z() → computes weighted sum

Training on MNIST (0 vs 1)

Binary Classification with Perceptron

from sklearn.datasets import fetch_openml
import matplotlib.pyplot as plt
import numpy as np

# Load MNIST
X, y = fetch_openml('mnist_784', version=1, return_X_y=True, as_frame=False)
y = y.astype(int)

# Filter 0 and 1
mask = (y == 0) | (y == 1)
X, y = X[mask], y[mask]
X = X / 255.0  # normalize

# Train perceptron
p = Perceptron(size=X.shape[1], learning_rate=0.01)
p.fit(X, y, epochs=10)

# Evaluate
preds = np.array([p.f(xi) for xi in X])
print("Accuracy:", np.mean(preds == y))

Perceptron with PyTorch

PyTorch Implementation

import torch
import torch.nn as nn
import torch.optim as optim
from torchvision import datasets, transforms

# MNIST (only digits 0 and 1)
transform = transforms.Compose([transforms.ToTensor(), 
                                transforms.Lambda(lambda x: x.view(-1))])

trainset = datasets.MNIST(root='./data', train=True, 
                          download=True, transform=transform)
mask = (trainset.targets == 0) | (trainset.targets == 1)
trainset.data, trainset.targets = trainset.data[mask], trainset.targets[mask]

trainloader = torch.utils.data.DataLoader(trainset, batch_size=32, shuffle=True)

# Perceptron model
class Perceptron(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc = nn.Linear(28*28, 1)

    def forward(self, x):
        return torch.sigmoid(self.fc(x))

model = Perceptron()
criterion = nn.BCELoss()
optimizer = optim.SGD(model.parameters(), lr=0.1)

# Training loop
for epoch in range(5):
    for data, target in trainloader:
        y = target.float().unsqueeze(1)  # match shape
        optimizer.zero_grad()
        output = model(data)
        loss = criterion(output, y)
        loss.backward()
        optimizer.step()

print("Training complete!")

Key Notes

  • ✅ nn.Linear(28*28,1) → perceptron layer
  • ✅ torch.sigmoid() → activation
  • ✅ BCELoss → binary classification loss
  • ✅ SGD → gradient descent optimizer

Implementing a Neural Network

A simple Feedforward Network Example 
import torch
import torch.nn as nn
import torch.optim as optim

# Define the model
class SimpleNN(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc1 = nn.Linear(3, 4)   # input: 3 → hidden: 4
        self.fc2 = nn.Linear(4, 2)   # hidden: 4 → output: 2
    
    def forward(self, x):
        x = torch.relu(self.fc1(x))  # activation function
        x = self.fc2(x)
        return x

# Instantiate model, loss, optimizer
model = SimpleNN()
criterion = nn.CrossEntropyLoss()
optimizer = optim.SGD(model.parameters(), lr=0.01)

⚡ This network has:

  • 3 input features → 4 hidden neurons → 2 outputs
  • ReLU activation for hidden layer
  • CrossEntropyLoss for classification
  • SGD optimizer for weight updates

What is a Tensor?

The fundamental data structure in PyTorch

Definition

A tensor is a multi-dimensional array with a single data type. It generalizes scalars (0D), vectors (1D), and matrices (2D) to higher dimensions (3D+). In PyTorch, tensors are like NumPy arrays but with support for GPU acceleration and automatic differentiation.

Tensor Ranks at a Glance

Rule of thumb: the number of dimensions is called the rank of the tensor.

Example dtypes: float32, int64, bool

Understanding
Conv2d in PyTorch

How convolution extracts features from an image
  • Input Image: The raw data (e.g., 28×28 pixels)
  • Kernel/Filter: Small weight matrix (e.g., 3×3) that slides over the image
  • Feature Map: Output of convolution highlighting patterns/features
  • Conv2d in PyTorch: nn.Conv2d(in_channels, out_channels, kernel_size)

Understanding
MaxPool2d

Reducing spatial dimensions while keeping key features
  • Input: Feature map from previous Conv layer
  • Window/Kernel size: typically 2×2 or 3×3
  • Stride: how far the window moves
  • Output: Smaller feature map keeping strongest signals
  • PyTorch: nn.MaxPool2d(kernel_size=2, stride=2)

Convolutional Neural Network (CNN)
with PyTorch

A Simple CNN for Image Classification 
import torch
import torch.nn as nn
import torch.nn.functional as F

class SimpleCNN(nn.Module):
    def __init__(self):
        super(SimpleCNN, self).__init__()
        self.conv1 = nn.Conv2d(1, 32, kernel_size=3, padding=1)
        self.pool = nn.MaxPool2d(2, 2)
        self.conv2 = nn.Conv2d(32, 64, kernel_size=3, padding=1)
        self.fc1 = nn.Linear(64 * 7 * 7, 128)
        self.fc2 = nn.Linear(128, 10)

    def forward(self, x):
        x = self.pool(F.relu(self.conv1(x)))   # Conv1 -> ReLU -> Pool
        x = self.pool(F.relu(self.conv2(x)))   # Conv2 -> ReLU -> Pool
        x = x.view(-1, 64 * 7 * 7)             # Flatten
        x = F.relu(self.fc1(x))                # FC1
        return self.fc2(x)                     # FC2 (logits)

This CNN architecture includes:

  • Conv layers: Feature extraction
  • Pooling layers: Dimensionality reduction
  • Fully connected layers: Classification

PyTorch Implementation:
Step 2: Train & Evaluate on MNIST 


from torchvision import datasets, transforms
import torch.optim as optim

# Load dataset
transform = transforms.Compose([transforms.ToTensor()])
train_loader = torch.utils.data.DataLoader(
    datasets.MNIST('./data', train=True, download=True, transform=transform),
    batch_size=64, shuffle=True)
test_loader = torch.utils.data.DataLoader(
    datasets.MNIST('./data', train=False, transform=transform),
    batch_size=1000, shuffle=False)

# Training & evaluation
device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
model = SimpleCNN().to(device)
optimizer = optim.Adam(model.parameters(), lr=0.001)

for epoch in range(1, 3):
    model.train()
    for data, target in train_loader:
        data, target = data.to(device), target.to(device)
        optimizer.zero_grad()
        output = model(data)
        loss = F.cross_entropy(output, target)
        loss.backward()
        optimizer.step()
    print(f"Epoch {epoch}: Loss={loss.item():.4f}")

# Evaluate
model.eval()
correct = 0
with torch.no_grad():
    for data, target in test_loader:
        data, target = data.to(device), target.to(device)
        pred = model(data).argmax(dim=1)
        correct += pred.eq(target).sum().item()

print(f"Test Accuracy: {100. * correct / len(test_loader.dataset):.2f}%")
  • Train on MNIST dataset.
  • Use Adam optimizer & cross-entropy loss.
  • Evaluate on test set → accuracy in %.

torch.save(model.state_dict(), "simplecnn.pth")

# Reload the model
model = SimpleCNN()
