    Coding Development with AI Workshop¶

    Table 12 ตารางการอบรมวันที่ 6-8 พฤษภาคม 2569 (รวม 18 ชั่วโมง)¶

    วันที่ / เวลา

    ประเภทกิจกรรม

    หัวข้อและรายละเอียดกิจกรรมสำคัญ

    วันที่ 6 พ.ค. 69

    09:00 - 11:00

    บรรยาย (2 ชม.)
    AI Foundations & Cost ROI

    • กลไกการทำงานของ LLM (Tokens, Context Window),

    • การคำนวณความคุ้มค่า (ROI) ของโปรเจกต์ และการประเมินต้นทุน Token ต่อฟีเจอร์

    11:00 - 12:00

    Workshop (1 ชม.)
    Environment Setup

    • การติดตั้งและกำหนดค่าเครื่องมือหลัก (Cursor / Windsurf) เพื่อเริ่มต้นการพัฒนาแบบ AI-native

    13:00 - 16:00

    Workshop (3 ชม.)
    Project Feasibility

    • ฝึกการวิเคราะห์ความเป็นไปได้ทางเทคนิค

    • การวางโครงสร้างโครงการเบื้องต้นโดยใช้ AI ร่วมสนับสนุน

    วันที่ 7 พ.ค. 69

    09:00 - 11:00

    บรรยาย (2 ชม.)
    Spec-Based Development

    • ศิลปะการเขียน Specification (7 Elements) เพื่อให้ AI สร้างโค้ดได้ถูกต้อง 100%

    • หลักการ Responsible & Ethical AI

    11:00 - 12:00

    Workshop (1 ชม.)
    Feature Building (Part 1)

    • การสร้างฟีเจอร์แรก (CRUD) จาก Spec ที่เขียนร่วมกับ AI โดยเน้นความแม่นยำของโครงสร้างโค้ด

    13:00 - 16:00

    Workshop (3 ชม.)
    Feature Building (Part 2)

    • การพัฒนาฟีเจอร์ที่ซับซ้อนขึ้น เช่น Authentication

    • การจัดการ State ภายในแอปพลิเคชัน

    วันที่ 8 พ.ค. 69

    09:00 - 11:00

    บรรยาย (2 ชม.)
    Advanced Architecting & Governance

    • การออกแบบ Database Schema และ API Contract

    • การกำหนดมาตรฐานทีมผ่านไฟล์ .clinerules เพื่อควบคุมพฤติกรรม AI

    11:00 - 12:00

    Workshop (1 ชม.)
    Database & API Lab

    • การใช้ AI ช่วยวิจารณ์ Architecture

    • สร้าง Entity-Relationship (ER) Diagram พร้อม API Specs โดยอัตโนมัติ

    13:00 - 16:00

    Workshop (3 ชม.)
    Governance Implementation

    • การเขียน Full Project Spec

    • การตั้งค่ามาตรฐานความปลอดภัยเพื่อเตรียมความพร้อมสำหรับช่วง Hackathon (Phase 2)

    สรุปรวมจำนวนชั่วโมงช่วงที่ 1

    • ชั่วโมงบรรยายรวม 6 ชั่วโมง

    • ชั่วโมง Workshop รวม 12 ชั่วโมง

    Table 13 ตารางช่วง Hackathon และ Pitching วันที่ 12-15 พฤษภาคม 2569 (รวม 21 ชั่วโมง)¶

    วันที่ / เวลา

    ประเภทกิจกรรม

    หัวข้อและรายละเอียดกิจกรรมสำคัญ

    วันที่ 12 พ.ค. 69

    Hackathon (Day 1)
    Core Backend & Architecture

    • เริ่มต้นพัฒนา MVP โดยเน้นการใช้ AI Agent ออกแบบโครงสร้างระบบและฐานข้อมูลตาม Spec ที่วางไว้

    09:00 - 12:00

    ปฏิบัติการ (3 ชม.)
    Project Initiation

    • การสร้าง Boilerplate และการกำหนดโครงสร้าง API สื่อสารระหว่าง Backend และ Frontend โดยใช้ AI

    13:00 - 16:00

    ปฏิบัติการ (3 ชม.)
    Database Implementation

    • การสร้าง Schema และการจัดการความสัมพันธ์ของข้อมูล (Data Relations) ผ่านการสนับสนุนของ AI

    วันที่ 13 พ.ค. 69

    Hackathon (Day 2)
    Frontend Integration & Logic

    • พัฒนาส่วนหน้าบ้านและการเชื่อมต่อลอจิกที่ซับซ้อนภายในแอปพลิเคชัน

    09:00 - 12:00

    ปฏิบัติการ (3 ชม.)
    UI/UX Implementation

    • การสร้าง Component และการจัดการ UI State โดยเน้นความเร็วและคุณภาพโค้ดจาก AI

    13:00 - 16:00

    ปฏิบัติการ (3 ชม.)
    Feature Completion

    • พัฒนาฟีเจอร์สำคัญ (Core Functionality) ให้ครบถ้วนตามความต้องการพื้นฐานของระบบ

    วันที่ 14 พ.ค. 69

    Hackathon (Day 3)
    Testing, Security & Deployment

    • การทดสอบความถูกต้องและการนำแอปพลิเคชันขึ้นใช้งานจริงบน Cloud

    09:00 - 12:00

    ปฏิบัติการ (3 ชม.)
    Automated Testing & Security

    • ใช้ AI ช่วยเขียน Unit Test และการทำ Automated Security Patching เพื่อลดความเสี่ยงของระบบ

    13:00 - 16:00

    ปฏิบัติการ (3 ชม.)
    Cloud Deployment

    • การนำแอปพลิเคชัน Deploy ขึ้นแพลตฟอร์ม Vercel หรือ Railway เพื่อทดสอบการใช้งานจริงในสภาพแวดล้อม Production

    วันที่ 15 พ.ค. 69

    Pitching (Day 4)
    Final Presentation & Feedback

    • การนำเสนอผลงานและรับฟังข้อเสนอแนะ

    09:00 - 12:00

    นำเสนอ (3 ชม.)
    Functional MVP Showcase

    • การ Demo ระบบที่ใช้งานได้จริง (Fully Functional) พร้อมสรุปบทเรียนและสถิติการใช้งาน AI ในโปรเจกต์

    สรุปรวมจำนวนชั่วโมงช่วงที่ 2

    • ชั่วโมง Hackathon (12-14 พ.ค.) 18 ชั่วโมง

    • ชั่วโมง Pitching (15 พ.ค.) 3 ชั่วโมง
